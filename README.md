# react-native-highcharts-wrapper

Highcharts react native wrapper

## Installation

```sh
npm install react-native-highcharts-wrapper
```

## Usage

```js
import { multiply } from "react-native-highcharts-wrapper";

// ...

const result = await multiply(3, 7);
```
